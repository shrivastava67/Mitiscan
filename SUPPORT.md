# Support

## Questions and Discussion

- **GitHub Discussions** — <https://github.com/shrivastava67/Mitiscan/discussions>
- **Bugs** — open an [issue](https://github.com/shrivastava67/Mitiscan/issues/new/choose).
- **Security** — see [SECURITY.md](SECURITY.md). **Never** report security
  issues in public issues or discussions.

## Before Filing a Bug

- Search existing issues and discussions.
- Try the latest `main`. Many bugs are already fixed.
- Run `mitiscan --check-deps` and include the output.
- Include OS, Python version, and the command that triggered the bug.

## Commercial Support

This project is community-maintained. There is no paid support tier today.
If you need enterprise SLAs, open a discussion describing your needs.
